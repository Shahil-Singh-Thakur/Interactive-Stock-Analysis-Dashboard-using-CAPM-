# 📊 CAPM Stock Analysis Dashboard

An interactive financial analytics web application built using **Python** and **Streamlit** to evaluate stock performance using the **Capital Asset Pricing Model (CAPM)**.

---

## 🚀 Overview

This project applies the CAPM model to estimate the expected return of stocks based on their systematic risk (beta). It allows users to fetch real-time market data, analyze stock behavior, and visualize risk-return relationships through an intuitive dashboard.

---

## 📈 Features

* 📌 Calculate **Expected Return** using CAPM
* 📊 Compute **Beta (β)** using historical stock data
* 🔄 Fetch real-time stock data via Yahoo Finance API
* 📉 Visualize stock price trends and performance
* 🧮 Analyze relationship between **risk and return**

---

## 🛠️ Tech Stack

* **Python**
* **Pandas**
* **NumPy**
* **Streamlit**
* **Plotly**
* **yfinance**
* **pandas-datareader**

---

## ⚙️ How It Works

1. Fetches historical stock and market index data
2. Computes daily returns
3. Calculates covariance and variance
4. Derives **Beta (β)**
5. Applies CAPM formula to estimate expected return
6. Displays insights via interactive charts

---


## 📌 Key Insights

* Higher beta stocks tend to have higher volatility and return potential
* CAPM helps evaluate whether a stock is **overvalued or undervalued**
* Visual analysis improves understanding of market behavior
